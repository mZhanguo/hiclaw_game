## 一、单局结构

### 1.1 基础参数

| 参数 | 数值 | 说明 |
|------|------|------|
| **单局总回合数** | 24回合 × 最多5阶段 = 最多120回合 | 实际大多数玩家在2-3阶段结束 |
| **单回合预计操作时间** | 1-2分钟 | 选择行动+处理事件+查看状态 |
| **单阶段时长** | 24-48分钟 | 含考评和过场 |
| **新手首局预期时长** | 30-45分钟 | 通常在县令阶段第2-3次考评翻车 |
| **熟练玩家通关时长** | 2-3小时 | 打到尚书阶段 |
| **平均单局时长** | 45-60分钟 | 大多数局在知府或布政使阶段结束 |

### 1.2 单阶段节奏

每24回合（1个官职阶段）分为3个8回合周期：

```
回合 1-8:   【建设期】 熟悉环境、认识NPC、建立基础
            → 第8回合：首次考评（容错高，40分即过）

回合 9-16:  【发展期】 深化关系、应对事件、积累优势
            → 第16回合：中期考评（标准难度）

回合 17-24: 【冲刺期】 事件密度加大、NPC暗战激化、全力备考
            → 第24回合：终期考评（决定晋升）
```

### 1.3 事件密度曲线

| 回合区间 | 每回合事件概率 | 事件类型倾向 |
|---------|-------------|------------|
| 1-4 | 30% | 教学型（引导性的简单选择） |
| 5-8 | 50% | 混合型（有好有坏） |
| 9-16 | 60% | 挑战型（两难抉择居多） |
| 17-20 | 70% | 高压型（连锁事件、NPC冲突） |
| 21-24 | 80% | 决战型（直接影响考评的大事件） |

## 二、死亡/失败设计

### 2.1 失败方式

| 失败方式 | 触发条件 | 频率预期 |
|---------|---------|---------|
| 考评不达标 | 考评总分<40 | 最常见（约60%的失败） |
| 民变 | 民望=0持续2回合 | 偶发（约15%） |
| 弹劾 | 周御史类NPC信任<-60 + 有把柄 | 中频（约15%） |
| 财政崩溃 | 储备=0且税收=0 | 罕见（约10%） |

### 2.2 失败并非终点

失败时进入**结算画面**，展示：
1. 本局存活回合数
2. 最高官职
3. 获得的Meta货币（官运点）
4. 解锁的永久内容

## 三、Meta成长系统

### 3.1 Meta货币：官运点

**获取方式：**

| 来源 | 数量 | 说明 |
|------|------|------|
| 存活每回合 | 1点 | 基础奖励，哪怕第1回合就死也有1点 |
| 通过考评 | 10点/次 | 鼓励活得久 |
| 考评优异 | 额外15点 | 鼓励打得好 |
| 考评卓异 | 额外25点 | 高手奖励 |
| 晋升 | 30点/次 | 里程碑奖励 |
| 达成成就 | 5-50点 | 一次性奖励 |
| 首次到达新官职 | 100点 | 大里程碑，仅限首次 |

**典型单局收益：**

| 玩家水平 | 预期存活 | 预期官运点 |
|---------|---------|-----------|
| 新手首局 | 16回合，未通过第2次考评 | 16 + 10 = **26点** |
| 新手第3局 | 24回合，通过县令晋升 | 24 + 30 + 30 = **84点** |
| 中等玩家 | 48回合，知府阶段结束 | 48 + 60 + 60 + 30 = **198点** |
| 高手通关 | 120回合，尚书通关 | 120 + 150 + 120 + 150 + 400 = **940点** |

### 3.2 Meta解锁树

#### 🌿 第一层：基础强化（0-100点总投入）

| 解锁项 | 花费 | 效果 |
|--------|------|------|
| 经验之谈·一 | 15点 | 初始民望+5 |
| 经验之谈·二 | 15点 | 初始治安+5 |
| 经验之谈·三 | 15点 | 初始税收+5 |
| 囊中羞涩 | 20点 | 初始储备+30 |
| 识人之明 | 35点 | 游戏开始时可查看1个NPC的性格参数 |

> **首局结算即可解锁1-2项**，让玩家感受到"下一局会更强"。

#### 🌳 第二层：机制解锁（100-400点总投入）

| 解锁项 | 花费 | 效果 |
|--------|------|------|
| 耳目通达 | 50点 | 解锁NPC关系网可视化 |
| 左右逢源 | 60点 | 社交行动好感+20%效果 |
| 开源节流 | 60点 | 每回合维护费用-2 |
| 运筹帷幄 | 80点 | 每3回合获得1点额外AP（本回合有效） |
| 未雨绸缪 | 70点 | 考评前1回合预览当前分数 |
| 东山再起 | 80点 | 单局1次，考评不达标时自动改为"勉强达标" |

> **玩3-5局后解锁核心机制**，改变玩法体验。

#### 🌲 第三层：高级特性（400-1000点总投入）

| 解锁项 | 花费 | 效果 |
|--------|------|------|
| 天命所归 | 120点 | 初始AP+1（每回合7点） |
| 八面玲珑 | 100点 | 所有NPC初始好感+10 |
| 铁腕手段 | 100点 | 解锁"威慑"行动（消耗信任换取强制执行） |
| 帝师传承 | 150点 | 解锁新起始角色"书生"（初始信任高） |
| 江湖侠义 | 150点 | 解锁新起始角色"侠客"（初始治安高） |
| 商贾世家 | 150点 | 解锁新起始角色"商人之子"（初始储备高） |

> **玩10-15局后逐步解锁**，提供长期追求目标。

#### 🏔️ 第四层：终极解锁（1000+点总投入）

| 解锁项 | 花费 | 效果 |
|--------|------|------|
| 命运之轮 | 200点 | 每局开始可选择1个"天命"buff |
| 青史留名 | 300点 | 解锁"自由模式"（无考评压力，纯剧情体验） |
| 权倾朝野 | 500点 | 解锁隐藏结局路线 |

### 3.3 解锁节奏感受

| 局数 | 累计预期官运点 | 可解锁层级 | 玩家感受 |
|------|-------------|-----------|---------|
| 第1局 | 26 | 第1层1-2项 | "下一把我更强了！" |
| 第2局 | 60 | 第1层3-4项 | "属性起点越来越高" |
| 第3局 | 144 | 第1层全部+第2层开始 | "终于能看到NPC关系网了！" |
| 第5局 | 340 | 第2层大部分 | "额外AP改变了节奏" |
| 第8局 | 600 | 第2层全部+第3层开始 | "新角色！换个玩法" |
| 第12局 | 1000 | 第3层大部分 | "可以挑战通关了" |
| 第15局 | 1400 | 第3层全部+第4层开始 | "追求隐藏结局" |
| 第20局+ | 2000+ | 全部解锁 | "自由模式随便玩" |

> **核心节奏：每1-2局必有新解锁。** 永远不会出现"打了一局什么都没变"的空虚感。

## 四、难度曲线设计

### 4.1 阶段内难度（以县令为例）

```
难度
  ▲
  │         ╭───╮
  │        ╱     ╲    ╭────╮
  │   ╭───╯       ╲  ╱      ╲
  │  ╱              ╲╱        ╲───╮
  │ ╱  建设期        发展期      冲刺期
  │╱
  └──────────────────────────────────▶ 回合
  1    4    8    12   16   20   24
```

- **回合1-4：** 低难度，引导性事件，属性容易提升
- **回合5-8：** 中低难度，首次考评压力，但阈值宽松（40分）
- **回合9-12：** 中等难度，负面事件增多，NPC开始有自己的agenda
- **回合13-16：** 中高难度，多线事件并发，资源紧张
- **回合17-20：** 高难度，关键NPC冲突爆发，必须做取舍
- **回合21-24：** 最高难度，最终考评前的密集考验

### 4.2 跨阶段难度膨胀

| 阶段 | 考评达标线 | 事件强度系数 | NPC阴谋复杂度 | 综合难度 |
|------|-----------|------------|-------------|---------|
| 县令 | 40/60/60 | ×1.0 | 简单直接 | ★★☆☆☆ |
| 知府 | 45/65/65 | ×1.3 | 两方博弈 | ★★★☆☆ |
| 布政使 | 50/70/70 | ×1.6 | 三方角力 | ★★★★☆ |
| 巡抚 | 55/75/75 | ×2.0 | 派系斗争 | ★★★★☆ |
| 尚书 | 60/80/80 | ×2.5 | 朝堂暗战 | ★★★★★ |

> 三个数字分别对应第1/2/3次考评的达标线。

### 4.3 难度调节机制

#### 隐性难度调整（玩家无感知）

| 条件 | 调整 |
|------|------|
| 连续2局在同一阶段失败 | 该阶段事件概率-10%，正面事件权重+15% |
| 首次到达新阶段 | 前4回合事件强度-20%（适应期） |
| 属性全部>70 | 下回合触发高难度事件概率+25%（防止碾压） |
| 存活超过80回合 | 随机事件趋向极端（高风险高回报） |

#### 显性难度选项（Meta解锁后开放）

| 难度 | 修正 | 官运点倍率 |
|------|------|-----------|
| 🟢 安逸 | 衰减-50%，考评阈值-10 | ×0.6 |
| 🟡 标准 | 无修正 | ×1.0 |
| 🔴 严苛 | 衰减+30%，考评阈值+5 | ×1.5 |
| 💀 地狱 | 衰减+60%，考评阈值+10，无借贷保护 | ×2.5 |

> 难度选项在完成首次通关（或累计10局）后解锁。

## 五、随机性控制

### 5.1 随机性来源

| 来源 | 影响程度 | 可控性 |
|------|---------|--------|
| 事件抽取 | 高 | 低（但Meta可解锁预览） |
| NPC行为 | 中 | 中（好感/信任高则行为可预测） |
| 考评细节 | 低 | 高（公式透明，可计算） |
| 行动结果 | 低 | 高（大部分行动是确定性的） |

### 5.2 防挫败设计

| 机制 | 说明 |
|------|------|
| **事件池去重** | 同一局内，同类型负面事件最多出现2次 |
| **连续保护** | 不会连续3回合都触发负面事件 |
| **翻盘机会** | 考评前最后1回合必定出现1个高收益正面选项 |
| **死亡预警** | 属性低于20时UI变红+文字提示 |
| **最低保障** | 即使全程摆烂，前16回合也不会直接Game Over |

### 5.3 种子系统

- 支持自定义种子（分享挑战局）
- 每日挑战：固定种子，全球排行榜
- 种子决定：事件序列、NPC性格微调、随机数序列
- 种子不决定：玩家选择带来的后果分支

## 六、单局体验目标总结

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 首局存活时间 | 15-30分钟 | 足够体验核心玩法但不至于太长 |
| 首次通关时间 | 第5-8局 | 有足够Meta积累后可首次通关县令 |
| 首次打到知府 | 第3-5局 | 给中期目标感 |
| 首次全通关 | 第15-25局 | 长期目标 |
| "再来一局"冲动 | 每局结束时 | 通过即时的Meta解锁反馈实现 |
| 单次游玩Session | 2-3局（1-2小时） | 适合碎片时间+深度时间 |
