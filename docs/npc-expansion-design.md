# 《官途浮沉》扩展NPC设计文档

> 策划：老K | 版本：v1.0 | 日期：2026-04-06

---

## 设计目标

现有NPC（9个）全部集中在县令阶段，刺史和侍郎阶段无专属NPC。本次扩展：

1. **刺史阶段新增3个NPC**——覆盖军政、地方豪族、朝廷耳目三条线
2. **侍郎阶段新增3个NPC**——覆盖朝堂派系、宦官势力、皇族外戚三条线
3. 每个NPC附带**背景故事、性格、派系、冲突/合作点**
4. 设计**3条NPC专属剧情线**，增加叙事深度

### NPC-P01：长史 裴行俭

> *"律令之下无私情，裴某一生只认理，不认人。"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_changshi_pei` |
| **官职** | 州长史（从五品上），实为刺史副手 |
| **派系** | Reformist（改革派） |
| **Rank** | Prefect |
| **Traits** | Righteous, Stubborn, Intelligent |
| **初始好感** | 5 |
| **初始信任** | -5 |
| **出场回合** | 25（刺史阶段开始） |
| **StartsActive** | false（晋升刺史后激活） |

**背景故事：**

裴行俭出身河东裴氏旁支，虽有名门血统，却因家道中落而走上科举之路。明经及第后，在地方辗转二十余年，从录事参军做到长史。此人精通唐律，案牍之功无人能及，但性情极为执拗——凡事只讲律令，不通人情。

上任刺史曾因他"不懂变通"而将他架空，但裴行俭岿然不动，暗中记录了上任刺史的十七条违法事实，最终在御史台巡查时一举扳倒旧主。由此在州府中人人忌惮——此人既是最好的帮手，也是最危险的下属。

**性格特征：**

- **刚直不阿：** 你的任何违法操作他都会记在心里。信任度低于-40时会直接向御史台检举。
- **认理不认人：** 好感度对他的行为影响较小，信任度才是关键——他尊重能力，不吃人情饭。
- **律令狂人：** 涉及司法、税务的决策卡中，选择合法选项会大幅加他的信任；选择灰色地带则大幅减。

**与玩家的冲突/合作点：**

| 情境 | 冲突可能 | 合作可能 |
|------|---------|---------|
| 军饷挪用 | 若你挪用河堤款发军饷，他会提出异议 | 若你据实上报，他会帮你拟写措辞严谨的奏折 |
| 盐铁私采 | 一旦发现你暗中采盐，信任直接降到底 | 若你正规上报，他会全力协助设立盐监 |
| 科场舞弊 | 他坚持重新开考，绝不妥协 | 若你彻查科场，他提供法律支持，考评大加分 |

**反应权重：**

### NPC-P02：团练使 韩令坤

> *"文官管得了案牍，管得了刀枪吗？这世道，拳头硬的说了算。"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_tuanlian_han` |
| **官职** | 团练使（正六品上），掌管州兵 |
| **派系** | Military（军方） |
| **Rank** | Prefect |
| **Traits** | Ambitious, Loyal（对军方）, Hot-tempered |
| **初始好感** | 0 |
| **初始信任** | -10 |
| **出场回合** | 25 |
| **StartsActive** | false |

**背景故事：**

韩令坤出身行伍，十六岁从军，在与吐蕃的边境战争中立下战功，从伙长一路升至团练使。此人武艺超群、治军有方，但完全不懂、也不屑于文官那套弯弯绕绕。

他与折冲都尉秦虎是旧交——两人曾在陇右并肩作战，生死之交。但秦虎如今是节度使的人，韩令坤却是朝廷直接委任的州兵长官，两人之间既有袍泽之情，也有军权之争。

韩令坤最大的心病是"门第之见"——他瞧不起科举出身的文官，认为天下是武人打下的，凭什么让书生来指手画脚。但内心深处，他又渴望得到文官系统的认可和尊重。

**性格特征：**

- **暴脾气：** 连续两次令他不满（好感变化为负），他会当众顶撞你，导致下属信任动摇。
- **重义气：** 一旦你赢得他的信任（>50），他会在关键时刻出兵相助——军事类危机卡自动降低一个难度等级。
- **武人骄傲：** 如果你在与他打交道时展现武事知识（选择Martial类选项），好感+额外加成。

**与玩家的冲突/合作点：**

| 征兵令 | 若你以老弱充数敷衍军方，他绝不原谅 | 若你全力配合征兵，他欠你一个大人情 |
| 流民叛乱 | 他主张血腥镇压，你若安抚招降他会嘲笑你 | 若你敢亲自上阵，他会刮目相看 |
| 藩镇压力 | 他受秦虎的影响，可能倒向藩镇 | 若你能说服他"朝廷才是正统"，他将成为最坚定的盟友 |

### NPC-P03：转运判官 柳如烟

> *"银子不会说谎。一州的账簿里，藏着所有人的秘密。"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_zhuanyun_liu` |
| **官职** | 转运判官（正七品上），负责本州漕运和盐铁转运 |
| **派系** | Neutral（表面），实际是宦官集团的暗线 |
| **Rank** | Prefect |
| **Traits** | Cunning, Greedy, Charming |
| **初始好感** | 15 |
| **初始信任** | -15 |
| **出场回合** | 27 |
| **StartsActive** | false |

**背景故事：**

柳如烟是个谜。她自称出身江南柳氏，以荫补入仕，但无人能查实她的家世。此人容貌端丽，谈吐风雅，精于财务——经她之手的每一笔账都滴水不漏。

真相是：柳如烟是宦官高力安插在地方的耳目。她负责监控各州财税流向，同时暗中为宦官集团截留一部分漕运利润。但她并非简单的走狗——她有自己的野心，想要在文官和宦官两个世界之间找到一条属于自己的路。

她对玩家的态度取决于利益——如果玩家有利用价值，她会主动靠近，提供情报和财务支持；一旦玩家成为威胁，她会毫不犹豫地出卖。

**性格特征：**

- **两面人：** 好感度和信任度经常反向变动。她可以对你很热情（好感高），但暗中向高力汇报你的一切（信任低）。
- **财务天才：** 如果你让她管钱，每回合税收+10%。但你的财务隐私对宦官集团完全透明。
- **可策反：** 如果你的好感度>70且信任度>50，她会告诉你自己的真实身份，并提出"双重间谍"的合作——这是解锁她专属剧情线的关键。

**与玩家的冲突/合作点：**

| 情境 | 冲突可能 | 合作可能 |
|------|---------|---------|
| 漕运改道 | 她暗中操纵漕船沉没（E-009的幕后黑手之一） | 若你与她合作，漕运码头建设成本降低30% |
| 盐铁之利 | 她会诱导你走"先采后报"的灰色路线 | 若你拒绝并上报，她反而会尊重你 |
| 宦官矫诏 | 她可能泄露你的行动给高力 | 若已策反，她会成为对付宦官的关键内线 |

**反应权重：**

| ActionType | FavorReaction | TrustReaction |
|------------|---------------|---------------|
| Politics | 1 | -2 |
| Social | 5 | 3 |
| Intelligence | 2 | -3 |
| SelfImprove | 0 | 0 |
| Scheme | 3 | 2 |

### NPC-V01：中书舍人 崔玄靖

> *"门阀之所以千年不衰，不是靠刀枪，而是靠血脉、姻亲和记忆。你一个寒门子弟，能在长安活过三代吗？"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_zhongshu_cui` |
| **官职** | 中书舍人（正五品上），负责起草诏令 |
| **派系** | Conservative（保守派/世族派） |
| **Rank** | ViceMinister |
| **Traits** | Cunning, Proud, Cultured |
| **初始好感** | -5 |
| **初始信任** | -15 |
| **出场回合** | 49（侍郎阶段开始） |
| **StartsActive** | false |

**背景故事：**

崔玄靖出身清河崔氏——唐代"五姓七望"之首。崔家在朝中盘踞数百年，宰相出过十余位，从不把科举新贵放在眼里。崔玄靖本人才华出众，文章冠绝同侪，二十三岁中进士，被誉为"崔家百年来第一才子"。

但他的才华掩盖不了骨子里的傲慢——他打心眼里瞧不起非世族出身的官员，认为治理天下是"诗书传家"的世族使命，寒门暴发户不过是朝堂上的过客。他是"李党"的中坚力量，主张恢复门荫入仕的传统，反对科举扩招。

崔玄靖最可怕的武器不是明枪，而是暗箭——他掌管中书省的诏令起草权，一字之差就能改变一道政令的走向。许多看似"措辞不当"的诏书，其实都是他的手笔。

**性格特征：**

- **世族傲骨：** 如果你出身寒门（默认设定），初始好感额外-10。但如果你通过Meta系统解锁了世族出身，初始好感+15。
- **文学品位：** 在文化类事件中（诗会、经筵）选择高品位选项，他会不自觉地欣赏你。
- **阴柔手段：** 他不会正面冲突，而是通过篡改诏令措辞、延误公文传递等方式给你使绊子。信任度低于-30时触发"公文延误"debuff——你的政策执行效率-20%。

**与玩家的冲突/合作点：**

| 情境 | 冲突可能 | 合作可能 |
|------|---------|---------|
| 科举改革 | 他是最坚定的反对者，会发动世族集团抵制 | 若你支持渐进改革，他愿意"给个面子" |
| 牛李党争 | 他是李党核心，你若加入牛党则水火不容 | 若你加入李党，他将成为你最有力的盟友 |
| 吏部铨选 | 他会施压安排世族子弟 | 若你给崔家面子，他在诏令起草上帮你说话 |
| 经筵进讲 | 若你引经据典不够精准，他会在朝会上公开纠正让你难堪 | 若你学识令他折服，他会破例为你起草加封诏书 |

### NPC-V02：神策军中尉 仇士良

> *"宦官怎么了？天子身边，离天子最近的，永远是我们。"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_shence_qiu` |
| **官职** | 神策军中尉（从三品），实控禁军 |
| **派系** | Eunuch（宦官集团） |
| **Rank** | ViceMinister |
| **Traits** | Cunning, Vengeful, Ruthless |
| **初始好感** | 0 |
| **初始信任** | -20 |
| **出场回合** | 49 |
| **StartsActive** | false |

**背景故事：**

仇士良，唐代宦官政治的终极代表。此人入宫四十年，侍奉过四位天子，亲手操纵过两次皇位更替。他以神策军中尉的身份掌控禁军——这支名义上保卫皇宫的军队，实际上是宦官手中的刀。

仇士良不识字，不懂经义，但他对权力的嗅觉超过朝堂上所有文官。他知道每个人的弱点，手里握着半个长安城的把柄。他对文官群体有一种本能的警惕和敌意——历史上每次"清除宦官"的尝试，都是文官发起的。

以历史上真实的仇士良为原型，他是"甘露之变"（H-004）中的最终BOSS。但在日常相处中，他并非无法打交道——只要你不触碰他的核心利益（神策军控制权），他甚至可以是一个"有用的朋友"。

**性格特征：**

- **极端记仇：** 好感度一旦跌破-30，永远无法回到正数。他会持续给你使绊子，直到你被贬或死亡。
- **贪婪可利用：** 他爱钱如命。送礼的好感加成是其他NPC的2倍，但信任度不会因此增加。
- **禁军威慑：** 如果你与他的信任度>30，朝堂上所有针对你的弹劾成功率-50%（谁敢得罪神策军？）。但一旦翻脸，弹劾成功率反而+30%。

**与玩家的冲突/合作点：**

| 情境 | 冲突可能 | 合作可能 |

| 宦官矫诏 | 你调查矫诏就是在挖他的根基，不死不休 | 若你假装没看到，他欠你一个天大的人情 |
| 甘露之变 | 如果你站在天子一边，他会调动禁军追杀你 | 如果你事先通风报信，他保你一世富贵 |
| 牛李党争 | 他暗中操控两党，谁不听话就扶持另一派 | 若你愿意做他在文官中的代理人，升迁之路一帆风顺 |
| 吏部铨选 | 他会安插自己的人 | 给他三个名额，剩下的你自己做主 |

### NPC-V03：驸马都尉 郑颢

> *"公主之婿，天子之臣，两者之间的距离，比长安到安西还远。"*

| 字段 | 内容 |
|------|------|
| **NpcId** | `npc_fuma_zheng` |
| **官职** | 驸马都尉（从四品下），兼任秘书监 |
| **派系** | Imperial（皇族/外戚） |
| **Rank** | ViceMinister |
| **Traits** | Melancholy, Cultured, Conflicted |
| **初始好感** | 10 |
| **初始信任** | 5 |
| **出场回合** | 52 |
| **StartsActive** | false |

**背景故事：**

郑颢本是荥阳郑氏的才子，状元及第后名满天下。然而命运弄人——天子看中他的才貌，将万寿公主下嫁于他。在别人看来这是天大的荣耀，对郑颢而言却是一座金笼：他心中早有婚约的卢氏女子被迫退婚；作为驸马，他在朝堂上的每一步都被视为"外戚干政"的嫌疑。

郑颢内心矛盾重重。他有治世之才，却因驸马身份而处处受限；他对公主无感，对卢氏念念不忘；他渴望在朝堂上有所作为，却只能以"秘书监"这个闲职打发日子。

他是朝堂上最特殊的存在——既是天子的至亲，又是最孤立的局外人。所有派系都对他保持距离——宦官忌惮他的皇族身份，文官嫉妒他的驸马特权，而他自己却两头不靠。

**性格特征：**

- **忧郁文人：** 在文化事件中对他展现真挚的友情（而非功利性社交），好感加成翻倍。
- **皇族身份：** 他是唯一可以直接面圣的NPC——如果你与他关系够好（好感>60），他可以帮你递折子绕过中书省（崔玄靖的封锁）。
- **外戚困境：** 他不愿被任何派系利用。如果你试图拉他站队，信任度直降-15。但如果你帮他解决个人困境（卢氏剧情线），他将对你掏心掏肺。

**与玩家的冲突/合作点：**

| 情境 | 冲突可能 | 合作可能 |

| 朝堂论战 | 他不会公开发言，但如果你的立场威胁到皇族利益，他会私下警告你 | 他可以在天子耳边替你说好话——一句顶万言 |
| 甘露之变 | 他会在事变中保护公主，未必会管你的死活 | 如果好感够高，他会冒生命危险通知你提前撤离 |
| 牛李党争 | 他厌恶两党之争，若你卷入太深，他会与你疏远 | 他是超越派系的中间人，可以帮你在两党之间斡旋 |

### 剧情线 A：「铁面长史」——裴行俭的选择

**所属NPC：** 裴行俭（NPC-P01）  
**阶段：** 刺史阶段  
**总长度：** 3个关键节点，贯穿整个刺史任期

**节点一：旧案重提（回合28-30触发）**

> 裴行俭找到你，神色凝重。他发现了前任刺史贪墨的最后一笔赃款下落——藏在州府后院枯井下的八千两白银。按律应充入国库，但这笔钱如果用来修缮河堤，可以解决困扰本州三年的水患。

- **选项A：依律上缴国库** → 裴行俭信任+20，但你失去一大笔可用资金
- **选项B：挪用修河堤，事后补手续** → 裴行俭信任-15（你在他面前弄虚作假），但民望+15，政绩+大
- **选项C：与他商量，请他想一个"合法"的办法** → 裴行俭好感+10（你尊重他），触发特殊对话："……容我想想。也许可以先以地方应急名义扣留，同时上报户部请示用途。律令中有此先例。"——解锁"合法挪用"选项

**节点二：执法困境（回合35-38触发）**

> 裴行俭在审理一桩土地纠纷时，查出涉案一方是他自己的远房堂兄。按他的原则，他应该回避。但他又担心其他人审理此案会偏袒另一方（韩令坤的心腹）。他罕见地来向你倾诉这个困境。

- **选项A：你亲自审理，让他回避** → 裴行俭信任+15，好感+10（你尊重了他的原则）
- **选项B：让他继续审，你信任他的公正** → 裴行俭好感+20（你对他的人格给予了最高评价），但韩令坤信任-10
- **选项C：将此案上交御史台** → 裴行俭好感-5（他觉得你不信任他），但信任+10（程序上无可挑剔）

**节点三：生死抉择（回合44-48触发，刺史末期考评前）**

> 藩镇叛军的前锋已到达州境。裴行俭连夜拟好了一份文书——将州府所有律令档案打包运往后方安全之处。他说："大人，我这一生只做了一件事——记录公道。这些文书比我的命重要。但如果大人决定降敌，我……另有打算。"

- **选项A：你们一起守城** → 裴行俭好感+30, 信任+30（你赢得了他一生的忠诚），触发战斗事件
- **选项B：你留守，让他护送文书撤退** → 裴行俭好感+20（你理解了他的执念），信任+20，但你失去一个战力
- **选项C：一起撤退，文书来日再说** → 裴行俭好感-20, 信任-30（他觉得你背叛了自己的承诺），可能当场决裂

**剧情线结局：**

- **铁血忠臣线（3个节点都选择尊重他的原则）：** 裴行俭成为你一生的盟友，在你晋升侍郎后推荐你进入御史台兼任，考评永久+10
- **裂痕线（1-2个节点违背他的原则）：** 关系降至"同僚"，公事公办，不再有额外帮助
- **决裂线（3个节点都违背他的原则）：** 裴行俭辞官归隐前，留下一封信——信中详列了你任内的所有违法事实。这封信被存放在御史台的密档中，成为悬在你头顶的达摩克利斯之剑

### 剧情线 B：「双面谍影」——柳如烟的秘密

**所属NPC：** 柳如烟（NPC-P03）  
**阶段：** 刺史阶段 → 可延伸至侍郎阶段  
**总长度：** 4个关键节点

**节点一：示好（回合28-32触发）**

> 柳如烟在一次宴席上主动接近你，言谈间展现出惊人的财务才能。她提出：如果你让她参与州府财务管理，她可以在不增税的前提下将岁入提升20%。她的方案详尽可行，但你的长史裴行俭私下提醒你："此人来历不明，大人慎之。"

- **选项A：接受她的方案** → 税收+20%/回合，柳如烟好感+15，但裴行俭信任-5
- **选项B：婉拒，但保持来往** → 柳如烟好感-5，裴行俭信任+5
- **选项C：暗中调查她的背景** → 无即时效果，2回合后获得情报："柳如烟与内侍省有秘密书信往来"

**节点二：泄密（回合36-40触发）**

> 你发现自己上月做出的一项财务决策（挪用/贪墨/合法皆可）的详细内容，已通过秘密渠道传到了京城内侍省高力的案头。如果你此前调查过柳如烟，你会怀疑是她所为；如果没调查过，你只知道"有内鬼"。

- **选项A：正面质问柳如烟** → 她会否认，但好感-10，信任-20
- **选项B：设局试探，故意放出假消息** → Scheming ≥ 15，2回合后确认柳如烟就是泄密者，解锁节点三
- **选项C：不动声色，以后行事更加小心** → 无即时后果，但以后每做一个"灰色"决策，宦官方面知悉概率+30%

**节点三：摊牌（接节点二，Scheming ≥ 15路线触发）**

> 你将证据摆在柳如烟面前。出乎意料的是，她没有狡辩，而是平静地说出了一切——她确实是高力的线人，但她同时也恨高力入骨。高力以她家人的性命相要挟，迫她做眼线。如今她愿意"反水"，条件是你保护她的家人。

- **选项A：接受她的投诚，暗中保护她的家人** → 柳如烟好感+30, 信任+25，解锁"双重间谍"模式——她将成为你对付宦官集团的王牌
- **选项B：将她的间谍身份上报朝廷** → 裴行俭信任+15，柳如烟好感-40, 信任-50（彻底敌对）
- **选项C：用她的秘密来要挟她为你做事** → Scheming +10，柳如烟好感-20, 信任-30，但她被迫服从

**节点四：终局（侍郎阶段，宦官矫诏/甘露之变相关事件触发时）**

> 如果你选了A路线：在对抗宦官集团的关键时刻，柳如烟冒死从宦官内部传出机密情报——高力的军事部署图。这份情报让你在"宫禁之变"（E-015）中多出一个隐藏选项："利用情报精确打击"（成功率最高的选项）。
> 
> 如果你选了B路线：柳如烟入狱后不久暴毙，留下一封血书——指控你才是宦官集团的真正内线。你需要花费大量精力自证清白。
> 
> 如果你选了C路线：柳如烟在关键时刻倒戈，向宦官出卖你的全部计划。

### 剧情线 C：「金笼中的状元」——郑颢的救赎

**所属NPC：** 郑颢（NPC-V03）  
**阶段：** 侍郎阶段  
**总长度：** 3个关键节点

**节点一：酒后真言（回合54-58触发）**

> 一次小型文人聚会后，郑颢独自留下。几杯酒下肚，这位驸马罕见地露出脆弱一面。他提到了那个名字——卢氏。"她至今未嫁，在洛阳白马寺旁开了间书斋，教孩子们读书。"他的眼中有光，也有痛。"公主待我不薄，可我……"话未说完，他苦笑摇头。

- **选项A：静静倾听，以友人身份陪伴** → 郑颢好感+20（他太久没有真朋友了）
- **选项B：劝他放下，驸马有驸马的责任** → 郑颢好感-5，信任+10（你说的是实话）
- **选项C：提议帮他暗中联络卢氏** → 郑颢好感+15, 信任+10，解锁节点二

**节点二：书斋来信（接节点一C路线，2-3回合后触发）**

> 你安排人暗中将郑颢的一封书信带到洛阳白马寺。数日后，回信到达——卢氏的笔迹，只有四个字："勿念，珍重。"但信封中夹着一片银杏叶。郑颢看到银杏叶时，久久无言，然后说："这是我们在太学时树下读书的那棵银杏。"

> 与此同时，万寿公主的侍女来报：公主近来多病，思虑过重，似乎对驸马的疏远有所察觉。

- **选项A：劝郑颢去陪伴公主，放下过去** → 郑颢好感-5（失望），但信任+15（你帮他看清了现实）。公主好感+暗中加成。
- **选项B：继续帮他与卢氏通信** → 郑颢好感+15，但风险积累——若被公主发现，将引发皇族震怒
- **选项C：利用这个秘密作为政治筹码** → Scheming +8，但郑颢好感-30, 信任-40（背叛）

**节点三：终局（回合68-72触发，侍郎末期）**

> 如果走A路线：郑颢在朝堂上公开为你说话，力排众议推荐你升任尚书。他在私下说："你是唯一一个既不利用我，也不可怜我的人。"他成为你在皇族中最坚定的支持者。

> 如果走B路线：公主终于发现了通信之事。天子震怒，下令彻查。郑颢被软禁，你被牵连——需要通过一系列高难度的自救操作（Scheming ≥ 20 / Charisma ≥ 20）才能脱身。若成功自救，郑颢在软禁中写了一首长诗，其中有一句："平生唯一知己，愧负亦不悔。"这首诗在民间广为流传，你的Reputation +15。

> 如果走C路线：郑颢在得知你利用他的秘密后，选择在朝堂上自陈其罪，公开自己与卢氏的通信，同时揭露你的勒索行径。你因"胁迫驸马"之罪被弹劾，考评大幅扣分。
